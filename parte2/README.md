# Parte 2 – Biblioteca API

Este projeto corresponde à **Parte 2** do trabalho prático da unidade curricular de Desenvolvimento Web.

O objetivo desta parte é o desenvolvimento de uma **API REST**, utilizando **Node.js**, **MySQL** e **Docker**, para a gestão de uma biblioteca.

A API permite realizar operações CRUD sobre autores, editoras, categorias e livros, incluindo relações do tipo **1:n**.
